# Мастер-годинник, це прилад який виступає "ведучим\керуючим" годинником для інших годинників в загальній мережі.

Реалізовані наступні параметри:
 - Керуючий імпульс для парної та непарної хвилини дорівнює 2 секунди
 - Для парної та непарної хвилини використані різні виводи однакової напруги (+5V)
 - При коригуванні поточного часу НАПЕРЕД через енкодер логіка  для керуючих імпульсів: 
    видача для кожної парної та непарної хвилини окремого імпульсу,
    де не парне значення(1..3..5..7). 
      Тобто на 5 хвилин світлодіоди: непарний, парний, непарний, парний, непарний.
 - По натисканню на кнопку енкодера включается режим вводу коррекції і 
    прокручуванням ручки виставляется  потрібне значення. 
    В разі корегування годинника назад імпульси НЕ ВИДАЮТЬСЯ (коррекція введенних годиників НАЗАД не потрібна).
 - Демонстрація температури з датчика DS3231 RTC module на екрані
 - Режим економії батареї - відключення екрану через 5 хвилин після натискання кнопки енкодера або інших.
 - Окрема коррекція майстер-годинника через кнопки BACK and CONFIG (на випадок розрядки батареі на DS3231)

 - В розробці робота годинника через переривання на контролері для більшої економії батареї
